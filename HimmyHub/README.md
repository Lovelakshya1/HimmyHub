## 🚀 Deploying to Vercel via GitHub

### Step 1 — Set up the repo

```bash
# If you have Git installed locally
git init
git add .
git commit -m "🌸 initial commit — HimmyHub"
```

Or simply upload `index.html` and `README.md` directly on [github.com](https://github.com/new).

> ⚠️ **Rename `himmyhub.html` to `index.html`** before uploading — Vercel needs this to serve it as the homepage.

### Step 2 — Push to GitHub

```bash
git remote add origin https://github.com/Lovelakshya1/himmyhub.git
git branch -M main
git push -u origin main
```

### Step 3 — Deploy on Vercel

1. Go to [vercel.com](https://vercel.com) and log in
2. Click **Add New → Project**
3. Click **Import** next to your `himmyhub` repo
4. Leave all settings as default — Vercel auto-detects static HTML
5. Hit **Deploy** 🚀

Your site will be live at `himmyhub.vercel.app` (or a custom domain if you set one).

### Auto-deploy on every push

Once connected, every `git push` to `main` will automatically redeploy. No manual steps needed.

```bash
# Making a change? Just do:
git add .
git commit -m "update: [what you changed]"
git push
```
